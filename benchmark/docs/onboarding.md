# Engineer Onboarding

New engineers should set up their local development environment on day one.
Clone the monorepo, run the bootstrap script, and request access to the
staging cluster through the internal access portal. Pair with a buddy for
the first two weeks to get familiar with the codebase and review process.

Complete the security training module before requesting production
credentials. Access to production systems is granted only after the
training is marked complete in the HR system.
